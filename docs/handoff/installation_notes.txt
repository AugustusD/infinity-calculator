===== FASCIA MEASURING TEMPLATE (Template_Measuring-Fascia-Infinity_Only.pdf) =====
Measurement fields:
A: Fascia height including decking
B: Drip edge, nosing or flashing
C: Gap from B to top of base plate
D: Base plate height if not standard* (standard = 5")
E: Offset if not standard* (standard 7/16", extended 1 1/2")
F: Blocking depth at fascia** (min 4.5")
G: Glass extension above post (2" typical) = top glass reveal
H: Rail height at glass from deck
I: Gap below glass to deck (2" typical) = bottom glass gap

Notes:
* Standard base plate height is 5" and standard offset is 7/16", extended offset is 1 1/2"
** Important: Please ensure topless rail along with fastening details are acceptable to local building authorities.
*** Glass is NOT included with the Infinity system

===== US vs CANADA RULES (from user specification) =====

SURFACE MOUNT:
US:
- Rail heights: 36 1/8" or 42 1/8" (nominal 36" or 42")
- Default top glass reveal: 2 1/8"
- Custom reveal: unlockable, range 0" to 2 1/8" (post gets taller as reveal decreases)
- Post top above deck = Rail Height - Top Reveal
- Min post top = Rail Height - 2 1/8" (standard), max = Rail Height (0" reveal)
- US: posts can be max 2" shorter than rail height → post top min = Rail Height - 2"
  So reveal range: 0" to 2" for US

CANADA:
- Rail heights: 36 1/8" or 42 1/8" (nominal 36" or 42")
- Post configurations: "Tall Post" or "Short Post"
- Tall post: post top finishes between 34" and 40" above deck
- Short post: post top finishes at minimum 24" above deck (at extreme: 42.125 - 18.125 = 24")
- At 42 1/8" rail height, max glass protrusion = 18 1/8" → post top = 42.125 - 18.125 = 24"
- If reveal increased such that post would go below 24", switch to short post system
- Custom reveal: unlockable, can be modified
- Min post height above deck: 24" (absolute minimum)

FASCIA MOUNT:
- Fascia post minimum overall length: 32"
- Physical fascia post length = (Rail Height - Top Reveal) + Distance_to_Deck + Base_Plate_Height
- Min physical length = 32" → (Rail Height - Top Reveal) + 3 + 5 ≥ 32"
  → Rail Height - Top Reveal ≥ 24" → Top Reveal ≤ Rail Height - 24"
  For 42 1/8": Top Reveal ≤ 18.125" (same as Canada surface constraint!)
  For 36 1/8": Top Reveal ≤ 12.125"

SETTING BLOCK > 5" RULE (FASCIA):
- If setting block height > 5": Replace setting block with wedge
- Use 1.5" piece of setting block
- Extra wedge needed at EACH side of post = (setting block space - 1.5")
- Setting block space = Bottom Gap + Distance_to_Deck
